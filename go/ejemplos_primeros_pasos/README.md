# 🚀 Ejemplos Básicos de Go

Este repositorio contiene ejemplos prácticos para aprender los conceptos fundamentales del lenguaje Go.

## 📚 Contenido

### Conceptos Básicos
- **ejemplo_1**: Variables y tipos de datos
- **ejemplo_2**: Funciones
- **ejemplo_3**: Bucles
- **ejemplo_4**: Arrays y slices
- **ejemplo_5**: Mapas
- **ejemplo_6**: Estructuras (structs)
- **ejemplo_7**: Interfaces
- **ejemplo_8**: Manejo de errores
- **ejemplo_9**: Punteros

### Complejidad Algorítmica (Big O)
- **bigo_1**: O(1) - Tiempo constante
- **bigo_2**: O(log n) - Tiempo logarítmico
- **bigo_3**: O(n) - Tiempo lineal
- **bigo_4**: O(n log n) - Tiempo lineal logarítmico
- **bigo_5**: O(n²) - Tiempo cuadrático
- **bigo_6**: O(2^n) - Tiempo exponencial

## 🛠️ Instalación

1. Descargar Go desde: https://go.dev/dl
2. Instalar siguiendo el asistente
3. Verificar instalación:

```bash
go version
```

## 🚀 Ejecutar ejemplos

Para ejecutar cualquier ejemplo, navega al directorio correspondiente y usa:

```bash
go run nombre_del_archivo.go
```

Por ejemplo:
```bash
cd ejemplo_1
go run variables_tipos.go
```

## 📖 Explicaciones

Cada ejemplo incluye comentarios en el código explicando los conceptos clave. Los ejemplos de Big O incluyen medición de tiempo para demostrar la complejidad en la práctica.
